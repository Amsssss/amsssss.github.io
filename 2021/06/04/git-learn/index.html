reset 几种模式

  • –soft: 移动head到指定位置， 不修改工作区和暂存区内容，与历史版本不一致的内容将添加到暂存区
  • –hard: 移动head到指定位置， 丢弃工作区，暂存区内容，完全切换到历史版本
  • –mixed: 移动head到指定位置， 不修改工作区，丢弃暂存区内容
